# 中国 AI 每周发展摘要 (CN AI Weekly Digest)

> 聚焦中国 AI 产业，每周筛选 TOP15 影响力事件，深度追踪技术突破、企业动态、政策监管、算力基建、应用落地五大维度。
>
> 方法论继承自 GitHub Weekly Curation 和 AI Weekly Digest (全球版) 的工程实践。

## 与全球版的差异

| 维度 | 全球版 (ai-weekly-digest) | 中国版 (cn-ai-weekly-digest) |
|------|--------------------------|------------------------------|
| **覆盖范围** | 全球 AI 全领域 | 中国 AI 产业 + 对华有直接影响的国际事件 |
| **事件数量** | TOP 20 | TOP 15 |
| **核心维度** | 6 维（基础模型/AI Agent/开源生态/政策监管/硬件算力/AI安全） | 5 维（大模型演进/应用落地/政策监管/算力自主/AI人才与资本） |
| **信息源** | 英文为主 + 中文交叉验证 | 中文为主 + 英文交叉验证 |
| **评分侧重** | 行业颠覆性 30% | 自主可控度 25%（新增维度） |
| **输出频率** | 每周一 | 每周一 |

## 五维影响力权重

| 维度 | 权重 | 说明 |
|------|------|------|
| **行业颠覆性** | 25% | 对中国 AI 产业格局的重塑程度 |
| **自主可控度** | 25% | 对国产替代、技术自主、供应链安全的意义 |
| **技术突破度** | 20% | 技术本身的新颖性和难度 |
| **生态影响力** | 20% | 对开发者/用户/下游生态的波及面 |
| **时间敏感性** | 10% | 事件的即时性和不可逆性 |

## 项目结构

```
cn-ai-weekly-digest/
├── .github/workflows/
│   ├── weekly-digest.yml       # 周一 09:00 CST 主工作流
│   └── watchdog.yml            # 周一 3 次补发检查
├── config/
│   ├── sources.yml             # 26 个信息源 + 6 个生态分组
│   └── keywords.yml            # 中文关键词（正/负向）
├── prompts/
│   ├── weekly-deep.md          # 深度分析 Prompt（中国版）
│   ├── taxonomy.md             # 五维分类 + 评分规则
│   └── feedback-rules.md       # 反馈闭环规则
├── db/
│   ├── schema.sql              # SQLite 表结构
│   ├── init_db.py              # 初始化 + 导入
│   └── query_db.py             # 查询 + 导出
├── scripts/
│   └── generate_context.py     # 生成上下文
└── output/
    └── YYYY-WNN.md              # 周报输出
```

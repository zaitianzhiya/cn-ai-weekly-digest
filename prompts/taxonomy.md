# 中国 AI 周报 — 五维分类与评分规则

## 主分类

| 分类 | 标签 | 典型特征 |
|------|------|---------|
| 大模型演进 | `#llm-evolution` | 新模型发布、架构创新、开源/闭源、Benchmark 突破 |
| 应用落地 | `#application` | 视频生成、AI Agent、自动驾驶、人形机器人、行业部署 |
| 政策监管 | `#policy` | 监管新规、出口管制、安全评估、备案制度、国家战略 |
| 算力自主 | `#chip-sovereignty` | 国产芯片、自研推理卡、算力集群、HBM、CUDA 替代 |
| AI人才与资本 | `#talent-capital` | 融资、IPO、估值、人才流动、收购、港股/A 股动态 |

## 影响力评分规则

### 正面信号（加分）
- 涉及头部企业（DeepSeek/智谱/字节/阿里/华为，+0.5）
- 有官方公告或政府文件支撑（+0.4）
- 对国产替代有直接推动（+0.5）
- 多源交叉验证（+0.3/额外源）
- 含量化数据（+0.3）

### 负面信号（减分）
- 单一来源（-0.5）
- 纯 PR 无实质内容（-0.4）
- 与上周重复（-0.3）

### 最终加权
```
基础分 + 信号调整 = 原始分
原始分 × Σ(维度权重 × 维度得分/10) = 最终权重
```

## "自主可控度"评分细则（中国版独有）

| 分数 | 标准 |
|------|------|
| 9-10 | 完全国产技术栈突破（如全栈国产芯片训练万亿模型） |
| 7-8 | 关键环节国产替代（如推理芯片自研、HBM 突破） |
| 5-6 | 生态适配进展（如适配国产 GPU） |
| 3-4 | 政策支持/引导国产化 |
| 0-2 | 与自主可控无关 |
